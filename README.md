# Medical Health Care Management System

A comprehensive web-based medical health care management system for hospitals, clinics, and healthcare facilities. This application provides an intuitive interface for managing patients, doctors, and appointments.

## Features

### 🏥 Dashboard
- **Real-time Statistics**: View total patients, today's appointments, available doctors, and pending appointments
- **Recent Activity**: Track recent appointments and system activities
- **Quick Overview**: Get insights into your healthcare facility's operations

### 👥 Patient Management
- **Add/Edit Patients**: Comprehensive patient registration with personal details
- **Search Functionality**: Quick patient lookup by name, email, or phone
- **Patient History**: Track patient visit history and medical records
- **Data Persistence**: All patient data is stored locally

### 👨‍⚕️ Doctor Management
- **Doctor Profiles**: Manage doctor information including specialties and experience
- **Specialization Tracking**: Organize doctors by medical specialties
- **Contact Information**: Store doctor's contact details and availability
- **Visual Cards**: Clean, card-based interface for doctor management

### 📅 Appointment Management
- **Schedule Appointments**: Easy appointment booking with patient-doctor matching
- **Status Tracking**: Monitor appointment status (scheduled, completed, cancelled)
- **Date/Time Management**: Flexible scheduling with date and time selection
- **Filtering Options**: Filter appointments by status and date
- **Quick Actions**: Mark appointments as complete or cancel as needed

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Custom CSS with responsive design
- **Icons**: Font Awesome 6.0
- **Storage**: Local Storage API for data persistence
- **Architecture**: Modular JavaScript classes with MVC pattern

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server setup required - runs entirely in the browser

### Installation
1. Clone or download the repository
2. Open `index.html` in your web browser
3. Start managing your healthcare facility!

### Usage
1. **Dashboard**: View overview statistics and recent activity
2. **Add Doctors**: Navigate to Doctors section and add medical staff
3. **Register Patients**: Use the Patients section to add patient information
4. **Schedule Appointments**: Create appointments linking patients with doctors
5. **Search & Filter**: Use search and filter features to find specific records
